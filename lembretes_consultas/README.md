# Sistema Automatizado de Lembretes de Consultas

## Descrição

Projeto desenvolvido para a disciplina Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas.

O sistema automatiza o envio de lembretes de consultas por e-mail e WhatsApp, ajudando clínicas e consultórios a reduzir faltas de pacientes.

## Funcionalidades

- Importação de consultas por planilha Excel
- Armazenamento em banco SQLite
- Envio de lembretes por e-mail
- Envio de lembretes por WhatsApp
- Controle de duplicidade
- Histórico de envios
- Geração de logs
- Agendamento automático

## Tecnologias Utilizadas

- Python 3
- SQLite
- OpenPyXL
- Schedule
- Twilio API
